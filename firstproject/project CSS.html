BeeShield — Intelligent Beehive Protection System

Concept:

BeeShield is a fictional smart device that protects beehives using sensors and alerts. It detects weather changes, predators, and temperature levels to keep the hive safe. The idea highlights how future tech could support beekeeping and ecosystems.

ROOM AI — Virtual Room Organization Assistant

Concept:

ROOM AI is an imaginary assistant that helps organize a bedroom using a simple virtual layout. It gives suggestions for furniture placement, lighting, and space optimization. The project mixes interior design with future smart-home technology.
